Why Jankx?

Jankx is not just a WordPress theme. It's a modern theme framework that brings Laravel-style architecture, Drupal-style content management, and React-style component thinking to WordPress.

Different Approaches, Different Strengths

WordPress themes serve different needs. Each approach has its place:

  • Page Builder themes (Divi, Elementor): Excellent for visual building, drag-and-drop convenience. Best for users who prefer GUI over code.
  • Starter themes (_s, Sage): Clean foundations for custom development. Great starting points that require building features from scratch.
  • Framework themes (Genesis): Proven architecture with extensive hooks. Established patterns that prioritize stability.

Jankx takes a different path - bringing modern PHP practices (DI containers, modular architecture) into WordPress theme development, with deep Gutenberg integration.

8 Unique Features

1. Layout Registry System

A dynamic layout management system with Dependency Injection support. Switch between Grid, List, Carousel, Masonry layouts without changing template files.

$registry->register('masonry', MasonryLayout::class);
$layout = $registry->resolve('grid', ['columns' => 3]);
echo $layout->render($query);

2. Extension System with Manifests

A full plugin-like extension system built into the theme. Extensions have lifecycle hooks, dependencies, and auto-activation.

3. Laravel-Style DI Container

A full Dependency Injection Container with Service Providers, Singletons, and Auto-resolution. Write clean, testable, loosely-coupled code.

$app->singleton(ViewService::class);
$layout = $app->make(GridLayout::class); // Auto inject

4. Deep Gutenberg Integration

Custom blocks: Swiper, Modal, SmartSearch, IconPicker, Typography, AdvancedFilters. Plus Block Extras - responsive visibility, child order controls.

5. JSON-Based Content Layout Patterns

Define complex layout patterns using JSON arrays instead of PHP templates. Non-developers can modify layouts. Version-control friendly.

6. Mobile Detection PRO

Server-side mobile detection that changes layout behavior, not just CSS. Load different content for mobile vs desktop.

7. Font & Icon Management PRO

Centralized font and icon management with visual pickers. Automatic Google Fonts loading, FontAwesome, Dashicons, Custom icons.

8. AssetResolver with Anti-CLS

Smart CSS/JS management with Critical CSS injection and priority loading. Eliminate Cumulative Layout Shift, improve Core Web Vitals.

Real-World Use Cases

  • News/Magazine: Multiple content types with different layouts (news grid, video carousel, list view)
  • E-commerce: Advanced product filtering with faceted search using AdvancedFilters block
  • SaaS/Product: Landing pages with modals, animations, icons
  • Multi-site: Shared features across 50+ sites with per-site customization

Who Should Use Jankx?

✅ Perfect for:

  • Developers building custom sites
  • Agencies with multiple projects
  • Teams needing maintainable code
  • Performance-focused projects

❌ Not for: Non-technical users wanting drag-drop, simple brochure sites
