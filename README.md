## 🚀 Quick Start / Bắt Đầu Nhanh

```bash
# Clone the repository
git clone https://github.com/jankx/jankx-lite.git

# Install dependencies
composer install
npm install

# Run tests
./vendor/bin/phpunit

# Build assets
npm run build
```
