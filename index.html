🚀 Quick Start / Bắt Đầu Nhanh

📥

Installation

Clone repository và cài đặt dependencies

git clone https://github.com/jankx/theme.git
cd theme
composer install
🧪

Testing

Chạy test suite để đảm bảo mọi thứ hoạt động

./vendor/bin/phpunit
🏗️

Development

Build assets cho production

npm install
npm run build
