Bắt Đầu với Jankx Theme

Chào mừng đến với Jankx Theme! Hướng dẫn này sẽ giúp bạn cài đặt và chạy theme một cách nhanh chóng.

Cài Đặt

Cách 1: Qua Composer (Khuyến Nghị)

composer require jankx/theme

Cách 2: Cài Đặt Thủ Công

  1. Tải bản release mới nhất từ GitHub Releases
  2. Giải nén vào /wp-content/themes/jankx/
  3. Kích hoạt trong WordPress Admin → Giao Diện → Themes

Cách 3: Git Clone

cd /wp-content/themes/
git clone https://github.com/jankx/jankx-lite.git jankx

Yêu Cầu

Yêu Cầu Phiên Bản
PHP>= 7.4
WordPress>= 5.8
Composer>= 2.0
Node.js>= 16.0 (cho development)

Cấu Hình

Thiết Lập Môi Trường

Tạo file .env trong thư mục gốc theme:

# Development Mode
JANKX_ENV=development
JANKX_DEBUG=true

# Cache Settings
JANKX_CACHE_ENABLED=false

# API Keys (tùy chọn)
GOOGLE_FONTS_API_KEY=your_key_here

Cấu Hình Theme

Chỉnh sửa config/app.php:

return [
    'providers' => [
        // Service Providers
        \Jankx\Support\Providers\ContentLayoutServiceProvider::class,
        \Jankx\Admin\Providers\AdminServiceProvider::class,
    ],
    'options' => [
        'framework' => 'jankx',
    ],
];

Các Bước Đầu Tiên

1. Cài Dependencies

composer install
npm install

2. Build Assets

# Development
npm run dev

# Production
npm run build

3. Chạy Tests

./vendor/bin/phpunit

4. Kích Hoạt Theme

Vào WordPress Admin → Giao Diện → Themes → Kích hoạt Jankx

Các Bước Tiếp Theo

  • Tìm hiểu về Hệ Thống Layout
  • Tạo Extensions Tùy Chỉnh

Xử Lý Lỗi

Lỗi: Class not found

# Giải pháp: Tạo lại autoload
composer dump-autoload

Lỗi: Styles không load

# Giải pháp: Build lại assets
npm run build

Lỗi: Permission denied trên cache

# Giải pháp: Sửa quyền
chmod -R 755 wp-content/themes/jankx/cache/
