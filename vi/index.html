Tài Liệu Jankx Theme

Chào mừng đến với tài liệu Jankx Theme. Đây là Phiên Bản Pro của theme WordPress framework hiện đại.

🚀 Bắt Đầu

Hướng dẫn cài đặt, thiết lập và bắt đầu nhanh cho người mới.

🏗️ Hệ Thống Layout

Layout động, block templates, và layout registry.

🧩 Extensions

Hệ thống extension, manifest, và quản lý vòng đời.

⚙️ Hệ Thống Quản Trị

Admin panels, form handlers, và theme options.

🧪 Kiểm Thử

Unit tests, integration tests, và code coverage.

📚 API Reference

Tài liệu API đầy đủ cho developers.

Bắt Đầu Nhanh

# Clone repository
git clone https://github.com/jankx/theme.git

# Cài đặt dependencies
composer install
npm install

# Chạy tests
./vendor/bin/phpunit

# Build assets
npm run build
